# 두 수의 합
가장 기초가 되는 문제이다.
```js
function solution(num1, num2) {
    const answer = (num1 + num2); //괄호는 없어도 된다. 구별을 위한 표기
    return answer;
}
```
다른 사람들은 어떻게 풀었을까?
```js
const solution = (num1, num2) => num1 + num2
```
함수가 한 줄이므로 화살표로 경량화한 모습이다.

```js
function solution(num1, num2) {
    var answer = 0;

    function add(a,b) {
        if (b==0) return a

        let sum = a^b;
        let carry = (a&b) << 1;

        return add(sum, carry);
    }

    return add(num1,num2);
}
```

# 두 수의 차
```js
function solution(num1, num2) {
    const answer = (num1 - num2);
    return answer;
}
```
