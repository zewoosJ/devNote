# 두 수의 나눗셈
```js
function solution(num1, num2) {
    var answer = parseInt(num1/num2*1000);
    return answer;
}
```

```js
function solution(num1, num2) {
    return Math.trunc(num1 / num2 * 1000);
}
```
Math.trunc라는 함수를 이용했다.
trunc는 소수점 이하 절사, floor는 소수점 이하 내림 이라는 차이점이 있다고 한다.
음수인 소수점을 없앨때는 trunc를 사용하는 것도 좋아보인다.
